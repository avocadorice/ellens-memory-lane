# 📸 Swapping in Your Real Photos

You can replace the cute placeholder sketches in the game with real photos of you and Ellen!

## 🚀 How to Add Your Photos

1. Crop your pictures to a square or roughly **1:1 aspect ratio** (e.g., `800x800` or `1000x1000` pixels) for the best fit inside the Polaroid frame.
2. Save your photos in this `photos/` folder.
3. Rename the files to match the expected names below:

| Level | Milestone | Filename |
| :--- | :--- | :--- |
| **1** | Graduation | `1_graduation.jpg` |
| **2** | Adopting Mochi | `2_mochi.jpg` |
| **3** | Engagement | `3_engagement.jpg` |
| **4** | Wedding Day | `4_wedding.jpg` |
| **5** | Our First Home | `5_first_home.jpg` |
| **6** | Welcoming Preston | `6_preston.jpg` |
| **7** | Our Second House | `7_second_home.jpg` |
| **8** | Welcoming Blaire | `8_blaire.jpg` |
| **9** | RV Camping | `9_camping.jpg` |
| **10** | Family at Mt. Fuji | `10_fuji.jpg` |

*(Note: The game expects `.jpg` files by default. If you use `.png` or another format, you can update the filenames in **[levels.js](file:///Users/bhsiao/dev/gdrive/bday/levels.js)**.)*

---

## 🎨 How it Works

The game preloads the images. If an image is missing or still loading, it falls back to drawing a beautifully stylized procedural colored sketch (e.g. key, baby bottle, campfire) so the game stays fully functional and artistic even without actual photo files!
