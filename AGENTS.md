# Project customizations (agy / Antigravity)

## Session start: morning recap

At the **start of a new session** in this workspace, before doing anything else,
run:

```
bash scripts/recap.sh
```

Then give me a short, friendly "here's where we left off" briefing based on its
output:

1. **The thread** — in 1–2 sentences, what was the active challenge / what we
   were in the middle of (infer from the recent prompts + uncommitted work).
2. **State** — branch, anything uncommitted, anything that looked unfinished.
3. **Next step** — the single most likely thing I'd want to pick up.

Synthesize — don't dump the raw log. I run many projects in parallel, so this is
just to jog my memory about where this one stands. After the briefing, ask what I
want to tackle.

I can also ask for "recap" at any time to re-run this.
